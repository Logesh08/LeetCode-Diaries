# [961. N-Repeated Element in Size 2N Array](https://leetcode.com/problems/n-repeated-element-in-size-2n-array/description/?envType=daily-question&envId=2026-01-02)

You are given an integer array <code>nums</code> with the following properties:

- <code>nums.length == 2 * n</code>.
- <code>nums</code> contains <code>n + 1</code> **unique**  elements.
- Exactly one element of <code>nums</code> is repeated <code>n</code> times.

Return the element that is repeated <code>n</code> times.

**Example 1:** 

```
Input: nums = [1,2,3,3]
Output: 3
```

**Example 2:** 

```
Input: nums = [2,1,2,5,3,2]
Output: 2
```

**Example 3:** 

```
Input: nums = [5,1,5,2,5,3,5,4]
Output: 5
```

**Constraints:** 

- <code>2 <= n <= 5000</code>
- <code>nums.length == 2 * n</code>
- <code>0 <= nums[i] <= 10^4</code>
- <code>nums</code> contains <code>n + 1</code> **unique**  elements and one of them is repeated exactly <code>n</code> times.
